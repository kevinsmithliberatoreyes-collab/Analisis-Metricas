# Análisis de Métricas de Rendimiento en Programación Paralela 

Este repositorio contiene la implementación en C de un algoritmo paralelo utilizando la API de **OpenMP**. El objetivo principal de este proyecto es calcular la suma de un arreglo masivo (100 millones de elementos) para analizar métricas clave de rendimiento en sistemas concurrentes: **Speedup**, **Eficiencia** y la aplicación de la **Ley de Amdahl**.

## Tecnologías utilizadas
* **Lenguaje:** C
* **API de Paralelización:** OpenMP


##  Cómo compilar y ejecutar el proyecto
